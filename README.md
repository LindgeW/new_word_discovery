## 基于无监督学习的词汇挖掘

相关任务：领域词汇（术语）挖掘、关键词抽取、新词发现、同义词挖掘、缩略词挖掘

### 统计特征
+ 频次
+ TF-IDF
+ C/NC-value：候选词的长度和嵌套性
+ PMI（自由凝固度）：词的内部凝聚程度
+ 左（右）邻字熵：词的左右邻字的丰富度
